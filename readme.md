TeamFinder

TeamFinder - веб-приложение на Django для поиска команды под pet-проекты.

Реализован вариант 1:
- избранные проекты;
- страница избранных проектов;
- фильтрация пользователей:
  - авторы избранных проектов;
  - авторы проектов, в которых я участвую;
  - пользователи, которым нравятся мои проекты;
  - участники моих проектов.

Стек:
- Python
- Django 5.2
- PostgreSQL
- Docker Compose
- Ruff
- Black

Запуск проекта локально:

1. Клонировать репозиторий:

git clone https://github.com/YanaKasaeva/team-finder-ad.git
cd team-finder-ad

2. Создать и активировать виртуальное окружение:

python3 -m venv venv
source venv/bin/activate

3. Установить зависимости:

pip install -r requirements.txt

4. Создать файл окружения:

cp .env_example .env

5. Запустить PostgreSQL:

docker compose up -d

6. Применить миграции:

python manage.py migrate

7. Создать суперпользователя:

python manage.py createsuperuser

8. Запустить сервер:

python manage.py runserver

После запуска проект доступен по адресу:

http://127.0.0.1:8000/

Админка:

http://127.0.0.1:8000/admin/

Основные страницы:

/                         - редирект на список проектов
/projects/list/           - список проектов
/projects/create-project/ - создание проекта
/projects/favorites/      - избранные проекты
/users/register/          - регистрация
/users/login/             - вход
/users/list/              - список пользователей
/users/change-password/   - смена пароля

Проверка проекта:

Django check:

python manage.py check

Тесты:

python manage.py test

Форматирование:

black users projects team_finder

PEP8 / lint:

ruff check users projects team_finder

Примечания:

В проекте используется шаблон варианта 1 из папки templates_var1.

Файл .env, виртуальное окружение, кеши и пользовательские медиафайлы не добавляются в репозиторий.
